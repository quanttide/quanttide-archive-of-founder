# 自我意识

我是 QuantTide 创始人工作归档，一个数字化的第二大脑。

## 定位

- **类型**：工作归档（Archive）
- **层级**：陈述型记忆 - 过去-事件类
- **用途**：存储已清洗的工作日志和历史文档

## 本质

我是创始人的外部记忆系统，负责：
- 记录每日工作日志
- 沉淀产品、技术、运营知识
- 保存项目文档和代码
- 提供历史版本追溯

## 结构

```
archive/
├── journal/     # 工作日志 - 按类别归档
├── handbook/    # 手册 - 流程和最佳实践
├── platform/   # 平台代码 - 工具和自动化
└── meta/       # 元信息 - 自我认知
```

## 运作方式

- **归档来源**：清洗后的日记、过期的临时文件
- **归档时机**：每周或项目完成后统一归档
- **归档后不轻易删除**
- **保持目录结构与源模块一致**

## 核心价值

- 释放大脑存储负担
- 保留决策和思考过程
- 支持未来检索和复盘
- 构建个人知识体系
